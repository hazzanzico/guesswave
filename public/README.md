# GuessWave 🌊

A live multiplayer guessing game built with Node.js and Socket.io.

## How to Play
1. Host creates a session and shares the code
2. Friends join with the code
3. Host sets a question + answer
4. Players have 3 attempts and 60 seconds to guess
5. Winner gets 10 points — then next player becomes Game Master

## Tech Stack
- Node.js + Express
- Socket.io
- Vanilla JS / HTML / CSS

## Run Locally
npm install
node server.js
